# Dodge the Fishes üêü

Welcome to **Dodge the Fishes**!
This is an endless arcade survival game developed with **Godot Engine 4**.

The goal is simple: dodge the fishes and survive as long as possible to set a new High Score!

---

## üõ†Ô∏è How to Install & Play
Since this is an open-source project, you can run the game directly inside the Godot Engine.

### Prerequisites
* You need **Godot Engine 4.x** installed on your computer. [Download Godot here](https://godotengine.org/download).

### Steps to Run:
1. **Download:** Click the green **<> Code** button above and select **Download ZIP** (or clone the repository).
2. **Extract:** Unzip the folder to your computer.
3. **Open Godot:** Launch the Godot Engine.
4. **Import:** Click the **Import** button and select the `project.godot` file inside the game folder.
5. **Play:** Press the **Play** button (top right corner) or press `F5` to start the game!

## üéÆ Features
* **Infinite Gameplay:** Enemies spawn endlessly with increasing difficulty.
* **3 Unique Enemies:**
    * **Shark:** Fast and large.
    * **Light:** Small and slow but tricky.
    * **Tartug:** Standard movement.
* **High Score System:** The game saves your best record locally.
* **Soundtrack & SFX:** Background music and game over effects.
