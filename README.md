# Dodge the Fishes üêü

An arcade game developed in Godot Engine.
Dodge the sharks, avoid the lights, and survive as long as possible!

## How to Play
1. **Download:** Go to the "Releases" section (or download the source code).
2. **Windows:** Extract the zip file and double-click the `.exe` file.
3. **Mac (Important):** - Extract the zip file.
   - **Do not** double-click the game immediately.
   - **Right-click** (or Control+Click) on the app and select **Open**.
   - Click **Open** in the confirmation dialog.

## Controls
- Use **Arrow Keys** (or WASD) to move.
- Avoid all enemies.
- Beat your High Score!
